\section*{Equipment Replacement Problem}
The equipment replacement problem is an optimization problem that involves deciding the optimal time to replace a piece of equipment (sell and re-buy) to minimize business costs. There are four main components to this problem:

\begin{itemize}
    \item The project term (e.g. 5 years long project)
    \item Life span of the equipment
    \item Price of new equipment
    \item Maintenance costs (could increase as equipment ages)
    \item Sell prices (e.g. 4 year old equipment can be sold for \$50)
\end{itemize}

The problem involves determining the optimal number of equipment replacements and the timing of each, in order to minimize operating costs.

Given the following variables:
\begin{itemize}
    \item $t_{max}$: Project term
    \item $t_{span}$: Lifes span of equipment
    \item $t$: A given instant through the project term.
    \item $C_{tx}$: The cost of buying at instant $t$ and selling at $x$
    \begin{itemize}
        \item Cost of new equipment + maintenance costs - sell price
    \end{itemize}
\end{itemize}

G(t) = optimal business costs from $t$ to $t_{max}$ (Buying new equipment at $t$).

\subsection{Trivial case}
G($t_{max}$) = 0
\begin{itemize}
    \item From instant $t_{max}$ to instant $t_{max}$
    \item We don't buy any equipment
\end{itemize}

\subsection{Optimization}
If new equipment is bought at $t$ we can sell it and buy new equipment at either $t + 1$, $t + 2$ , ..., or $t + t_{span}$. We should sell at the point where we get the least operation cost given the project term.

\subsection{Bellman Equation}
Knowing this, we can establish the Bellman equation:
\begin{equation}
    G(t) = \min \{C_{tx} + G(x)\} \\
\end{equation}
\begin{centering}
    From $t = t_{max}$ until $t = 0$.
\end{centering}

\subsection*{Step by Step Calculations}
$G(5) = 0$ \quad (Base case)

$G(4) = \min\{C_{45} + G(5) = (300 + 50 - 200) + 0 = 150\} = 150$

\textbf{Optimal choice:} sell at time 5

$G(3) = \min\{C_{34} + G(4) = (300 + 50 - 200) + 150 = 300, C_{35} + G(5) = (300 + 50 + 60 - 150) + 0 = 260\} = 260$

\textbf{Optimal choice:} sell at time 5

$G(2) = \min\{C_{23} + G(3) = (300 + 50 - 200) + 260 = 410, C_{24} + G(4) = (300 + 50 + 60 - 150) + 150 = 410\} = 410$

\textbf{Multiple optimal choices:} sell at time 3, sell at time 4 (tie)

$G(1) = \min\{C_{12} + G(2) = (300 + 50 - 200) + 410 = 560, C_{13} + G(3) = (300 + 50 + 60 - 150) + 260 = 520\} = 520$

\textbf{Optimal choice:} sell at time 3

$G(0) = \min\{C_{01} + G(1) = (300 + 50 - 200) + 520 = 670, C_{02} + G(2) = (300 + 50 + 60 - 150) + 410 = 670\} = 670$

\textbf{Multiple optimal choices:} sell at time 1, sell at time 2 (tie)
